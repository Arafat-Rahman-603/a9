# MediQueue – Tutor Booking System

**Live URL:** [Deploy on Vercel](#) 

MediQueue is a premium tutor booking web application where students can discover, register, and book online learning sessions effortlessly. Built with Next.js, it offers a seamless and modern user experience.

## Features
- **Dynamic Tutor Search & Filtering:** Quickly find tutors by name or session availability date.
- **Secure Authentication:** Integrated NextAuth for Email/Password and Google OAuth login.
- **Real-time Availability:** Smart slot booking system prevents double-booking and manages tutor availability automatically.
- **Dark/Light Mode Theme:** Beautiful, responsive UI with smooth theme toggling using DaisyUI.
- **Personalized Dashboards:** Manage your created tutoring sessions or track your booked learning sessions from dedicated private pages.
